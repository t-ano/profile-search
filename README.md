### 利用した技術

- Laravel 8　（PHPフレームワーク）
- Laravel Breeze　（認証機能パッケージ）
- SQLite　（データベース）
- Tailwind CSS　（CSSフレームワーク）

### 要件

- タレントのプロフィールを検索して見ることができる。
- お問い合わせができる。
- タレントの管理ができる。
- 会員登録、ログインができる。

### 機能

- ログイン画面（ユーザー、管理者）
- プロフィール一覧画面 + 検索機能
- プロフィール登録画面（管理者）
- 問合わせ画面
- 問合わせ一覧・詳細画面（管理者）
